About the Results

The performance results are extracted from Performance API. When the button is clicked to take performance measurement, two network resources are checked: one for library code (Data Collection Tag, at.js, Web SDK/alloy.js) and one for their API call to load personalization. Measurements represent network start and network end time in milliseconds. Their average number is highlighted for convenience. All results are stored in local storage - you can clear them when going to the next page. See more details in main.js.
