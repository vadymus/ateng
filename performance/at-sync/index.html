About the Results

The performance results are extracted from Performance API. When the `Run Performance Test` button is clicked, two network resources are checked: one for library code (Data Collection Tag, at.js, alloy.js (Web SDK)) and one for their API call to load personalization. Measurements represent the difference of network start and network end time in milliseconds. Their average number is highlighted for convenience. All results are stored in local storage - you can clear them when going to the next page. Keep browser's Dev Tools closed for cleaner results. See code details in main.js.
